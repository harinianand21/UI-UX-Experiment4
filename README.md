# UI-UX-Experiment4

## Aim
To design low-fidelity wireframes for a generic application using **Balsamiq**, demonstrating the basic layout structure, navigation flow, and placement of UI elements for key screens of a digital product.

---

## Algorithm

1. Identify the essential screens required for a basic application workflow.
2. Open **Balsamiq Wireframes** and create a new project.
3. Design the **Welcome / Splash Screen** with logo placeholder and a "Get Started" button.
4. Create the **Login / Signup Screen** with input fields for email and password along with login and signup options.
5. Design the **Home / Dashboard Screen** including navigation bar, search bar, category shortcuts, and content cards.
6. Create the **Content / Product Listing Screen** displaying multiple items with placeholders, titles, and action buttons.
7. Design the **Checkout / Final Action Screen** with order summary, address/contact input, and confirmation button.
8. Ensure the wireframes are **low-fidelity**, using only grayscale boxes and text placeholders without colors or graphics.
9. Maintain clear navigation flow between screens.
10. Export the wireframes as images and include them in the repository and lab report.
